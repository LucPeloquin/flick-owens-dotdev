WM4K reference assets

Selected textures in this app were derived from Alan-bur/WM4K:
https://github.com/Alan-bur/WM4K

The full WM4K repository is intentionally not vendored here. Runtime assets
under `public/assets/wii/` are either direct copies of selected textures or
small composed previews generated from those textures for this site's menu.

The main menu background uses WM4K's `Wii Menu/tex1_8x456...` strip repeated
horizontally, matching how the original UI composes some large surfaces from
small textures.
