# Wii Sound Drop-ins

Drop files here using the exact names expected by `lib/audio/sounds.ts`.

## Required SFX filenames

- `tink.mp3`
- `back.mp3`
- `select.mp3`
- `zoom.mp3`
- `unzoom.mp3`
- `hover.mp3`
- `disc-insert.mp3`
- `home-open.mp3`
- `home-close.mp3`
- `mii-pose.mp3`

## Optional BGM loop filenames

- `bgm-menu.mp3`
- `bgm-mii.mp3`
- `bgm-settings.mp3`
- `bgm-mail.mp3`

## Sourcing notes (research links)

- Wii BIOS/UI sounds index (community rip host):  
  https://www.sounds-resource.com/wii/systembioswii/sound/21664/
- Wii Menu UI sprite sheets/icons:  
  https://www.spriters-resource.com/wii/wiimenu/

Use assets you have rights to use. If you skip these files, the app falls back gracefully.
